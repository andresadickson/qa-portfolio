# Portfólio QA - Andresa

Este repositório foi criado para praticar testes manuais e começar a desenvolver minhas habilidades na área de QA.

---

## Objetivo

Aplicar na prática conceitos de testes, como:

- Criação de checklist
- Escrita de casos de teste
- Registro de bugs

---

## O que foi feito

Realizei testes manuais utilizando o site Mercado Livre como base, explorando funcionalidades como busca, login e navegação.

---

## Estrutura

- checklist-mercado-livre.md → checklist dos testes realizados
- test-case-login.md → teste de login com dados válidos
- test-case-login-invalido.md → teste com dados inválidos
- bug-mercado-livre.md → exemplo de bug report

---

## Aprendizados

Esse projeto foi meu primeiro contato prático com testes manuais. No início tive dificuldade em entender como estruturar os testes, mas aos poucos fui conseguindo organizar melhor as ideias e documentar de forma mais clara.

---

## Observação

Projeto desenvolvido para estudo e evolução na área de Quality Assurance.
