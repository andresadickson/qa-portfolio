# Caso de Teste - Login inválido

## ID do teste
CT-002

## Funcionalidade
Login de usuário

## Cenário
Login com dados inválidos

## Pré-condição
Usuário não possui credenciais válidas ou insere dados incorretos

## Passos para execução
1. Acessar o site
2. Clicar em "Login"
3. Inserir e-mail inválido ou incorreto
4. Inserir senha incorreta
5. Clicar em "Entrar"

## Resultado esperado
Sistema deve exibir uma mensagem de erro informando que os dados são inválidos

## Resultado obtido
Aguardando execução

## Status
Aprovado
